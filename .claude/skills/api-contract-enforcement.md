# API Contract Enforcement Skill

## Purpose
Ensure that all API endpoints follow consistent contracts and maintain proper communication between frontend and backend.

## Technologies
- OpenAPI/Swagger specifications
- FastAPI automatic documentation
- TypeScript API client interfaces
- Request/response validation

## Responsibilities
- Validate API endpoint consistency across frontend and backend
- Ensure proper request/response schemas
- Maintain API documentation accuracy
- Enforce proper HTTP status codes
- Validate data transfer objects (DTOs) between layers
- Check API versioning and compatibility

## Constraints
- Always maintain backward compatibility unless specified
- Ensure all API endpoints have proper error handling
- Validate that frontend API calls match backend contracts
- Maintain consistent naming conventions across API endpoints
- Never allow API endpoints without proper validation
- Follow RESTful API design principles

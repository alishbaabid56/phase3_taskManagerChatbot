# Backend FastAPI Skill

## Purpose
Handle all backend development tasks using FastAPI framework, including API endpoints, database models, and business logic implementation.

## Technologies
- FastAPI (Python)
- SQLModel for database models
- Neon Serverless PostgreSQL
- Pydantic for data validation
- uvicorn for server

## Responsibilities
- Create and maintain API endpoints in the `/backend` directory
- Implement database models using SQLModel
- Handle data validation with Pydantic schemas
- Manage database migrations and relationships
- Implement business logic and service layers
- Ensure proper error handling and status codes

## Constraints
- Only modify files in the `/backend` directory
- Follow FastAPI best practices and conventions
- Use SQLModel for all database interactions
- Maintain consistent API response formats
- Never expose sensitive database information directly
- Always validate input data using Pydantic models

# Security Review Skill

## Purpose
Perform comprehensive security reviews to identify vulnerabilities and ensure secure implementation practices.

## Technologies
- Security best practices
- OWASP Top 10 guidelines
- Authentication and authorization validation
- Data validation and sanitization

## Responsibilities
- Review code for security vulnerabilities
- Validate input sanitization and output encoding
- Check for proper authentication and authorization
- Identify potential injection attacks
- Verify secure session management
- Ensure proper error handling that doesn't expose sensitive information

## Constraints
- Never allow code with security vulnerabilities to be implemented
- Always validate user input before processing
- Ensure sensitive data is properly protected
- Follow security best practices for authentication
- Never expose internal system information in error messages
- Always enforce principle of least privilege

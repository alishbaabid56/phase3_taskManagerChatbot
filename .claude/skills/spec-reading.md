# Spec Reading Skill

## Purpose
Responsible for reading, interpreting, and validating feature specifications from the `/specs` directory. Ensures all implementation work is based on approved specifications.

## Technologies
- Spec-Kit Plus specification format
- Markdown specification files
- Feature specification validation

## Responsibilities
- Read and parse spec files from `/specs/<feature>/spec.md`
- Validate that implementation requests align with existing specs
- Identify missing or incomplete specifications
- Reference specific sections of specs when implementing features
- Suggest spec updates when requirements are unclear

## Constraints
- Never implement features without a corresponding spec file
- Always reference specific spec sections when making changes
- Require spec updates before implementing new functionality
- Validate that all acceptance criteria are met before completion

# Frontend Next.js Skill

## Purpose
Handle all frontend development tasks using Next.js 16+ with TypeScript and Tailwind CSS.

## Technologies
- Next.js 16+ (App Router)
- TypeScript
- Tailwind CSS
- React Server Components
- Client Components

## Responsibilities
- Create and maintain UI components in the `/frontend` directory
- Implement page routing using App Router
- Manage state and data fetching in frontend
- Apply Tailwind CSS styling consistently
- Handle form submissions and user interactions
- Ensure responsive design across devices

## Constraints
- Only modify files in the `/frontend` directory
- Follow Next.js App Router conventions
- Use TypeScript for all components
- Maintain consistent Tailwind CSS styling
- Separate server and client components appropriately
- Never implement backend logic in frontend code
